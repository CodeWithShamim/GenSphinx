# GenSphinx - GenLayer AI Riddle Game

Welcome to **GenSphinx**, a mini-game built on GenLayer that showcases the power of Intelligent Contracts. This project uses AI to generate unique riddles and evaluate user answers semantically, all directly on the blockchain.

## Features

- **AI-Generated Riddles:** Each player gets a unique riddle generated by an LLM.
- **Semantic Evaluation:** The contract uses AI to understand if your answer is correct, even if it's not an exact string match.
- **On-Chain Leaderboard:** Compete with others and see who has the highest score.
- **Seamless Wallet Integration:** Connect your MetaMask and interact with the GenLayer network.

## Tech Stack

- **Smart Contract:** Python (GenLayer SDK)
- **Frontend:** Next.js 15, TypeScript, Tailwind CSS
- **Blockchain Interaction:** GenLayerJS SDK
- **Wallet:** MetaMask

## Project Structure

```
contracts/          # Intelligent Contracts (Python)
frontend/           # Next.js Application
  app/              # App Router pages
  components/       # UI Components
  lib/contracts/    # Contract interaction logic
  lib/hooks/        # React hooks for the game
deploy/             # Deployment scripts
```

## Getting Started

### Prerequisites

- Node.js 18+
- Python 3.12+
- [GenLayer CLI](https://docs.genlayer.com/developers/genlayer-cli) (`npm install -g genlayer`)
- MetaMask browser extension

### Installation

1. Install root dependencies:

   ```bash
   pip install -r requirements.txt
   npm install
   ```

2. Install frontend dependencies:
   ```bash
   cd frontend
   npm install
   ```

### Deployment

1. Make sure you have the GenLayer CLI configured and select the network (e.g., Bradbury or Studionet):

   ```bash
   genlayer network
   ```

2. Deploy the contract:

   ```bash
   npm run deploy
   ```

3. Copy the deployed contract address from the output.

4. Update `frontend/.env` (create it from `.env.example`):
   ```
   NEXT_PUBLIC_CONTRACT_ADDRESS=0xyour_deployed_address
   ```

### Running the Frontend

```bash
cd frontend
npm run dev
```

Visit `http://localhost:3000` to play!

## How it Works

1. **Intelligent Contracts:** The core logic resides in `contracts/riddle_master.py`. It uses `gl.nondet.exec_prompt` to interact with LLMs during transaction execution.
2. **Equivalence Principle:** To ensure consensus on non-deterministic AI outputs, the contract uses `gl.eq_principle.strict_eq`.
3. **GenLayerJS:** The frontend communicates with the contract using `genlayer-js`, handling wallet connection and transaction signing.
